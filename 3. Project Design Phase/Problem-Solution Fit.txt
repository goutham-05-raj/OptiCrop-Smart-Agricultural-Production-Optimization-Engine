This deliverable maps user challenges to features of the Opticrop application:

1. User Problem: Mismatch between crops and localized soil/climate properties.
 - Solution Feature: Random Forest Classifier recommends optimal crops from 22 profiles by evaluating N, P, K, pH, temp, humidity, and rainfall.

2. User Problem: Over-application of expensive chemical fertilizers, causing soil damage.
 - Solution Feature: NPK Vitality Prescription Engine calculates deviations from optimal ranges and suggests fertilizer corrections.

3. User Problem: Inability to forecast yields, increasing financial risk.
 - Solution Feature: Random Forest Regressor predicts expected crop yield (t/ha) based on locations and seasonal weather patterns.

4. User Problem: Difficulty diagnosing soil category and health properties.
 - Solution Feature: Unsupervised K-Means clustering classifies soil into 5 diagnostic clusters, offering automated classifications.
