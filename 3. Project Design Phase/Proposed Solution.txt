The proposed Opticrop solution is an intelligent decision support system. Key features include:

1. Ensemble Machine Learning Predictions:
   - Random Forest Classifier recommends crops (100% test accuracy).
   - Random Forest Regressor predicts crop yields.
   - K-Means Clustering groups soil into 5 categories.
2. NPK Correction Engine:
   - Flags nutrient deficiencies/excesses and provides chemical corrective guidelines (e.g. Urea, SSP, MOP).
3. Analytics Telemetry Dashboard:
   - Visualizes model performance, feature importances, and confusion matrices to help users understand recommendations.
4. Responsive Frontend UI:
   - A dark-themed glassmorphism interface built with CSS and Bootstrap 5, providing a responsive experience across devices.
